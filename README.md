# recruitai-fastapi
FastAPI Backend of AI Recruiter Website

- **Intelligent Job Matching:** Uses OpenAI's language models to analyze resumes and job descriptions, delivering personalized job recommendations based on both candidate interests and qualifications with detailed match explanations.

- **Vector Database Integration:** Combines PostgreSQL, DuckDB and Pinecone for efficient storage and semantic search across thousands of job positions with configurable filters for salary, posting date and categories.

- **Resume Processing & Optimization:** Extracts text from PDF resumes and generates customized CV summaries and objectives tailored to candidate profiles and target positions.

- **Secure API Design:** Built with FastAPI featuring JWT authentication, dependency injection patterns, and comprehensive error handling for production-grade security and reliability
